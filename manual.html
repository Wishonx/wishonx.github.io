HTML šablona

V této šabloně najdete vše, co potřebujete pro začátek nové Grove Framework webové stránky. Šablona obsahuje všechny potřebné meta značky pro responzivitu webu. 


<!DOCTYPE html>
<html lang="en">
                    
<head>
    <!--Potřebné meta značky-->
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    
    <!--Grove Framework CSS-->
    <link rel="stylesheet" href="Grove/css/Grove_framework.css">
                    
    lttitle>Your Website</title>
</head>
<body>
    <!--Obsah stránky-->
                    
</body>
</html>

Grid

Grove Framework využívá fluidní grid systém o dvanácti sloupcích, který zajišťuje responzivitu webových stránek. Grove Framework také umožňuje nastavení breakpointů jednotlivých sloupců pro lepší a lehčí stylování mobilních zařízení. Syntaxe je velmi jednoduchá a vhodná i pro začínající vývojáře!

Jak grid funguje:

Pro použití gridu je nutné, aby jeho rodičovský element byl <div> třídy .container.

col-3
col-4
col-3

<div class="container">
    <div class="col-3">col-3</div> <!-- Tento sloupec zabírá 3/12 (1/4) z třídy container -->
    <div class="col-6">col-4</div> <!-- Tento sloupec zabírá 4/12 (1/3) z třídy container -->
    <div class="col-3">col-3</div> <!-- Tento sloupec zabírá 3/12 (1/4) z třídy container -->
</div>

Tlačítka

Stylování tlačítek s Grove Framework je velmi jednoduché. Framework také nabízí předpřipravená tlačítka pro rychlé užití. Samotný element <button> nic nedělá, a to aby stylování vlasního tlačítka nebylo zbytečně kontraproduktivní. Pro základní layout tlačítka je tu k dispozici předpřipravená třída .btn.


<button class="btn btn-light"> button-light </button>
<button class="btn btn-dark"> button-dark </button>
<button class="btn btn-1"> button-1 </button>
                            

<button class="btn btn-outline-light"> outline-light </button>
<button class="btn btn-outline-dark"> outline-dark </button>
<button class="btn btn-outline-1"> outline-1 </button>
                            

<button class="btn btn-outline-light btn-s"> small button </button>
<button class="btn btn-outline-light btn-m"> medium button </button>
<button class="btn btn-outline-light btn-l"> large button </button>
<button class="btn btn-outline-light btn-xl"> xlarge button </button>
                            

<button class="btn btn-outline-light "> default border </button>
<button class="btn btn-outline-light btn-sharp">sharp border </button>
<button class="btn btn-outline-light btn-round">round border </button>

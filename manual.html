HTML šablona

V této šabloně najdete vše, co potřebujete pro začátek nové Grove Framework webové stránky. Šablona především obsahuje všechny potřebné meta značky pro zajištění responzivity webu.

Pokud jste nevložili soubor Grove_framework do Vašeho kmenového souboru, nezapomeňte změnit cestu ke Grove_framework.css.


<!DOCTYPE html>
<html lang="en">
                    
<head>
    <!--Potřebné meta značky-->
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    
    <!--Grove Framework CSS-->
    <link rel="stylesheet" href="Grove_framework/css/Grove_framework.css">
                    
    lttitle>Your Website</title>
</head>
<body>
    <!--Obsah stránky-->
                    
</body>
</html>

Grid

Grove Framework využívá fluidní grid systém o dvanácti sloupcích, který zajišťuje responzivitu webových stránek. Grove Framework také umožňuje nastavení breakpointů jednotlivých sloupců pro lepší a lehčí stylování mobilních zařízení. Syntaxe je velmi jednoduchá a vhodná i pro začínající vývojáře!

Jak grid funguje:

Pro použití gridu je nutné, aby jeho rodičovský element byl <div> třídy .container.

col-3
col-4
col-3

<div class="container">
    <div class="col-3">col-3</div> <!-- Tento sloupec zabírá 3/12 (1/4) z třídy container -->
    <div class="col-6">col-4</div> <!-- Tento sloupec zabírá 4/12 (1/3) z třídy container -->
    <div class="col-3">col-3</div> <!-- Tento sloupec zabírá 3/12 (1/4) z třídy container -->
</div>

Tlačítka

Stylování tlačítek s Grove Framework je velmi jednoduché. Framework také nabízí předpřipravená tlačítka pro rychlé užití. Samotný element <button> nic nedělá, a to aby stylování vlasního tlačítka nebylo zbytečně kontraproduktivní. Pro základní layout tlačítka je tu k dispozici předpřipravená třída .btn.

Základní tlačítka

Pro vytvoření základního předpřipraveného tlačítka přiejte ke třídě .btn třídu .btn-{preset}, kde za {preset} vložíte Vámi vybraný preset:


<button class="btn btn-light"> btn-light </button>
<button class="btn btn-dark"> btn-dark </button>
<button class="btn btn-1"> btn-1 </button>

Obrysová tlačítka

Pro vytvoření obrysového tlačítka pouze stačí přidat outline- před vybraný preset: 


<button class="btn btn-outline-light"> btn-outline-light </button>
<button class="btn btn-outline-dark"> btn-outline-dark </button>
<button class="btn btn-outline-1"> btn-outline-1 </button>

Veikosti tlačítek

Pro výběr velikosti tlačítka stačí k tlačítku přidat třída .btn-{velikost}, kde za {velikost} vložíte vybranou veikost tlačítka:


<button class="btn btn-outline-light btn-s"> small button </button>
<button class="btn btn-outline-light btn-m"> medium button </button>
<button class="btn btn-outline-light btn-l"> large button </button>
<button class="btn btn-outline-light btn-xl"> xlarge button </button>

Okraje tlačítek

Základní okraj tlačítek je okraj se zakulacenými rohy. Pokud tedy chcete změnit okraj stačí přidat třídu .btn-sharp pro ostrý okraj nebo .btn-round pro kulatý okraj:


<button class="btn btn-outline-light "> default border </button>
<button class="btn btn-outline-light btn-sharp">sharp border </button>
<button class="btn btn-outline-light btn-round">round border </button>

Tabulky

Grove Framework také nabízí jednoduchou tabulku ve světlém a tmavém provedení. Základní elementy tabulky nejsou stylovány, a tak se to elementu <table> musí přidat třída .table-light nebo .table-dark. Doporučuje se také správné užívání značek <thead> a <tbody>.

Jméno Příjmení Bydliště
František Novotný Praha
Robert Vránský Liberec 

<table class="table-light">
    <thead>
        <tr>
            <th>Jméno</th> <th>Příjmení</th> <th>Bydliště</th>
        </tr>
    </thead>
    <tbody>

        </tr>
